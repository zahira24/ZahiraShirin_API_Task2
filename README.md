# REST API Data Fetcher & Dashboard 📊

## 🚀 Project Overview
This project fetches live data from a public API, processes it, and displays it in both table and graphical formats.

## 🛠️ Tech Stack
- Python 🐍
- requests (API handling)
- tabulate (table display)
- matplotlib (data visualization)

## 📌 Features
- Fetch live API data 🌐
- Parse JSON responses
- Display data in table format 📋
- Visualize data using charts 📊
- Error handling (timeouts, connection errors, invalid responses)

## 📂 How to Run
```bash
python main.py
